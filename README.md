# DW1000 STM32 Driver

## Описание
Библиотека для работы с DWM1000 UWB модулями на STM32F411RET6.
Предназначена для Модуля Коммуникации и Сопряжения (МКС) с интерфейсом MATLAB.

## Особенности
- Поддержка 2x DWM1000 на одном SPI
- Командный интерфейс для MATLAB
- Эталонный TX/RX для проверки эфира
- Полная поддержка регистров DWM1000
